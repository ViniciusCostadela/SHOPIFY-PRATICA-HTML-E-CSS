🛒 Clone da Interface do Shopify (Projeto Prático)
Este é um projeto de desenvolvimento web focado na replicação da interface visual (front-end) da página do Shopify. O objetivo principal foi aplicar na prática os conceitos teóricos que venho absorvendo ao longo do meu aprendizado na área de tecnologia.

🚀 Sobre o Projeto
O site foi construído do zero como um exercício prático para o curso técnico em desenvolvimento web que estou realizando. Através dele, pude treinar a estruturação de layouts complexos, alinhamento de elementos e responsividade.

🛠️ Tecnologias Utilizadas
HTML5: Para a estruturação semântica de toda a página.

CSS3: Para a estilização, trabalhando com cores, tipografia, espaçamentos e organização do layout.
